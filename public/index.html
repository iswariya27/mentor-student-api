Student - Mentor API

Students API Endpoints

  • Get all Students - /api/students
  • Get Student by ID - /api/student/${student_ID}
  • Post Student name with body { "name":${student_name} } - /api/student
  • Assign mentor to student and students to mentor using Student ID (Patch) with body { "mentorId":${mentor_id} } - /api/student/${student_ID}
  • Delete Student by ID - /api/student/${student_ID}

Mentors API Endpoints

  • Get all Mentors - /api/mentors
  • Get Mentor by ID - /api/mentor/${mentor_ID}
  • Post Mentor name with body { "name":${mentor_name} } - /api/mentor
  • Assign students to mentor by mentor ID (Patch) with body { "studsId":[${studId}] } - /api/mentor/${mentor_ID}
  • Delete Student by ID - /api/mentor/${mentor_ID}
